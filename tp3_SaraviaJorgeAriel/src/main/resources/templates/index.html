Programación Visual en Java

La programación visual utiliza interfaces gráficas para mejorar la interacción con el usuario y facilitar el desarrollo.

  • Swing: Una biblioteca para crear interfaces gráficas ricas en Java.
  • JavaFX: La evolución de Swing, con más características y una arquitectura moderna.
  • AWT: El conjunto de herramientas de widgets abstractos, la base para la creación de GUI en Java.
